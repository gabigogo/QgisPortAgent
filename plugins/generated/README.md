# plugins/generated/

This folder is the canonical output destination for generated or converted QGIS plugins.

Examples:
- plugins generated by migrate-arc
- plugins generated by new-plugin
- plugins generated by research/tbx_to_qgis.py

Generated plugin directories are intentionally gitignored to keep the repository clean.
Only this README is tracked.

Expected structure:

plugins/generated/
  <tool_name>_plugin/
    metadata.txt
    __init__.py
    main_plugin.py
    processing_provider/
    tests/
    migration_report.md

To link generated plugins into your QGIS profile on Windows:

```powershell
.\scripts\setup_plugins.ps1 -Category generated -IncludeLegacyGenerated
```

To link generated plugins on Linux/macOS:

```bash
bash scripts/setup_plugins.sh --category generated --include-legacy-generated
```
